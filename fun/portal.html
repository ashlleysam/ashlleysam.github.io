PortalAdvance Emulator

Play the PortalAdvance ROM in a browser using the gba-kit Game Boy Advance emulator.

Status: Initializing...

Controls:

  • Arrow keys — D-pad
  • Z — A button
  • X — B button
  • Enter — Start
  • Backspace — Select
  • A — R shoulder
  • S — L shoulder

If the emulator does not start automatically, press Run.

The source is available at github.com/ashlleysam/PortalAdvance.
